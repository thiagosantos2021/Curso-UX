Estruturando User Flows

Os fluxos de usuários podem ser criados à moda antiga (nova) com caneta e papel ou digitalmente, em sites como Flowmapp e Wireflow. Uma dica valiosa é usar algumas plataformas diferentes como teste para criar fluxos de usuários e descobrir uma que atenda às suas necessidades como UX designer.

Durante o processo de estruturação dos seus User Flows, responda às seguintes perguntas:

  • O que seu usuário está tentando realizar?
  • Quais informações o usuário precisa para realizar a tarefa?
  • Existem barreiras que o usuário enfrenta ao tentar concluir a tarefa em questão?

Nesta linha, seu fluxo de usuário mostrará os vários caminhos que um usuário pode tomar para atingir seu objetivo dentro do seu produto. Para mostrar isso, você usará formas e setas para indicar quando as informações são apresentadas, bem como diferentes ações tomadas e decisões tomadas pelo usuário. Cada etapa de ação será representada por uma forma.

Após criar o fluxo de usuário, você pode buscar feedback das partes interessadas e revisá-lo para ver se há alguma peça faltando. Então, você pode começar a wireframing e trabalhar no resto do processo de design de experiência do usuário.
